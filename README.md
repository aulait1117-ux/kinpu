# 菌譜（きんぷ / Kinpu）

臨床検査技師のための、抗菌薬耐性のノート帳。スマホだけで完結する。

菌を選ぶと、その1画面に3つがそろっている。

- **耐性ナビ** — 感受性の S / I / R を押すと、次にやるべき確認試験・報告のしかた・届出義務が出る
- **自然耐性** — 生まれつき効かない薬。ここが「感性」と出たら、菌種同定か測定を疑う
- **ノート** — 手書きノートの写真と、自分の言葉のまとめ。全文検索できる

RAIDAR と同じく GitHub Pages に置いた静的サイト。URLを開くだけで動く。サーバーは無い。

## データはどこにあるか

**この端末のブラウザの中だけ**（localStorage と IndexedDB）。外のサーバーには一切送らない。写真もアップロードしない。

裏返すと、ブラウザの履歴を消したり機種変すると消える。設定画面から JSON で書き出して、ときどきメールやクラウドに送っておく。

## 患者情報を入れない

保存時に、カルテID・氏名・生年月日・病棟番号らしき文字列を機械で検出して警告する。ただし完全ではない。**写真に検体ラベルや氏名が写り込んでいないか、撮った直後に自分で確認する。**

## 判定ルールは「下書き」である

`rules.js` に入っている確認試験のルールは、すべて `verified: false` で出荷している。画面上は「未確認 — 鵜呑みにしない」と表示される。

CLSI M100 の最新版と自施設のSOPで1件ずつ突き合わせ、確認できたものだけ「確認済み」にチェックを付ける。ブレークポイントの数値は年次改訂で変わるので、このアプリには載せていない。載せると古くなった瞬間に嘘になる。

医学的な中身は `rules.js` の1ファイルに集約してある。読んで直せる。

## 手書きノートの取り込み

OCRサーバーは持っていない。代わりにスマホのOS機能を使う。

- iPhone：写真を長押し → 「テキストを認識」→ コピーして本文に貼る
- Android：Google レンズ で写真を開く → テキストを選択してコピー

写真そのものはノートに添付できるので、文字起こしは要点だけでいい。

## 収録している菌（13）

黄色ブドウ球菌 / CNS / 腸球菌（フェカリス・フェシウム） / 肺炎球菌 / 大腸菌 / 肺炎桿菌 / 誘導型AmpC産生菌群 / プロテウス / 緑膿菌 / アシネトバクター / ステノトロフォモナス / インフルエンザ菌

足りない菌は画面から自分で追加できる（薬剤・自然耐性も自分で入れる）。

## 構成

| ファイル | 中身 |
|---|---|
| `index.html` | 画面とCSS（企業道OS カフェ・エディトリアル ダーク） |
| `rules.js` | 菌・自然耐性・確認試験の判定ルール（医学的中身はここだけ） |
| `app.js` | 保存・検索・判定・バックアップ |
